Récapitulatif Javascript

Objectifs :

  • révision des fondamentaux (conditions, boucles, valeurs et types)
  • conception de fonctions robustes et réutilisables
  • refactoring de code existant

Reprendre des exercices faits depuis le début de la formation (si possible, les reprendre tous !), et les améliorer, en créant des fonctions adaptées là où cela semble nécessaire. Il n'y a pas de "bonne réponse", mais l'idée générale est de simplifier la lecture de vos programmes, en masquant le code "pénible" (ex: vérifier qu'une chaine de caractère n'est pas vide) dans des fonctions.

En bas du fichier main.js, vous trouverez deux exercices supplémentaires :)

Les déclarations des fonctions doivent être faites dans le fichiers functions.js

Le code "réel" du/des programmes doit se trouver dans le fichier main.js
