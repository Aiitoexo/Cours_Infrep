Exercice 4

Ici, le code CSS est déjà écrit. Votre but est d'associer ces classes aux éléments qui en ont besoin.
Vous ne pouvez pas ajouter de balises HTML, ni modifier le css existant. Vous devez seulement trouver les classes qui peuvent satisfaire les exigences de chaque bloc. (il peut y avoir des fausses pistes !)

Toutes les classes ne sont pas forcément à utiliser, et certaines ne doivent pas être utilisées car elles contiennent des erreurs!!

Dans un second temps, quand tout fonctionne, essayer de "nettoyer" le code CSS .

  1. Supprimer le code CSS inutile/erroné
  2. Renommer les classes utilisées dont le nom n'est pas très explicite
  3. Eventuellement réorganiser le code, en regroupant les classes dans un ordre plus "logique", mais attention, souvenez vous que "le dernier qui parle à raison", donc il ne faut rien casser en bougeant du code !
